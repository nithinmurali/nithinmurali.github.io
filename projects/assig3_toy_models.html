Key Bindings:

Switch between Orthographic and perspective camera:

KEY_P

Rotate Camera around:

KEY_A, KEY_D, KEY_W, KEY_S, KEY_Q, KEY_E

To Switch between Humanoid-toy to Ant-toy

Press: KEY_T

To translate the selected toy, use the following keys:

KEY_J, KEY_L, KEY_I, KEY_K, KEY_U, KEY_O

To rotate the selected body part, use the following keys:

KEY_LEFT, KEY_RIGHT, KEY_UP, KEY_DOWN, KEY_PAGE_UP, KEY_PAGE_DOWN

Note: the elbows and knees of the humanoid and the lower legs of the ant have only one degree degree of rotation (KEY_UP and KEY_DOWN)

To rotate the whole toy, choose the root node before pressing rotation keys.

To choose a specific body part, refer the following key bindings.

For humanoid:

-------------------------------

KEY_0 = Select torso (root node)

KEY_1 = Select right lower leg

KEY_2 = Select head

KEY_3 = Select left upper arm

KEY_4 = Select right upper arm

KEY_5 = Select left upper leg

KEY_6 = Select right upper leg

KEY_7 = Select left lower arm

KEY_8 = Select right lower arm

KEY_9 = Select left lower leg

For ant:

-------------------------------

KEY_0 = Select torso (root node)

KEY_1 = Select head

KEY_2 = Select left middle upper-leg

KEY_3 = Select right middle upper-leg

KEY_4 = Select left front upper-leg

KEY_5 = Select right front upper-leg

KEY_6 = Select left back upper-leg

KEY_7 = Select right back upper-leg

KEY_8 = Select left middle lower-leg

KEY_9 = Select left front lower-leg

KEY_V = Select left back lower-leg

KEY_B = Select right middle lower-leg

KEY_N = Select right front lower-leg

KEY_M = Select right back lower-leg
